==========================================
  体重管理アプリ - サーバー起動方法
==========================================

■ サーバーの起動
  → start-server.bat をダブルクリック
  → 自動的にブラウザが開きます

■ サーバーの停止
  → stop-server.bat をダブルクリック

■ 注意事項
  ・サーバー起動中は黒いウィンドウを閉じないでください
  ・アプリを使い終わったら必ずサーバーを停止してください

■ トラブルシューティング
  ・「ポートが使用中」というエラーが出る場合
    → stop-server.bat を実行してから再度起動してください

  ・ブラウザが開かない場合
    → 手動で http://localhost:8000 にアクセスしてください
